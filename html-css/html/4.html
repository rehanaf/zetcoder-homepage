Text dan Tautan

Text Formatting

Text formatting dalam HTML melibatkan penggunaan elemen dan atribut untuk mengubah tampilan atau presentasi teks di halaman web. Berikut beberapa elemen HTML yang sering digunakan untuk melakukan format pada teks:

Bold (<b> atau <strong>)

Penjelasan: Membuat teks menjadi tebal.

contoh:


<p>This is <b>bold</b> text.</p>

Italic (<i> atau <em>)

Penjelasan: Membuat teks menjadi miring.

contoh:


<p>This is <i>italic</i> text.</p>

Underline (<u>)

Penjelasan: Memberikan garis bawah pada teks.

contoh:


<p>This is <u>underlined</u> text.</p>

Strikethrough (<s> atau <del>)

Penjelasan: Menyisipkan garis melintang pada teks.

contoh:


<p>This is <s>strikethrough</s> text.</p>

Superscript (<sup>) dan Subscript (<sub>)

Penjelasan: Mengangkat atau menurunkan teks dari garis dasar.

contoh:


<p>This is <sup>superscript</sup> text.</p>
<p>This is <sub>subscript</sub> text.</p>

Tautan

Dalam HTML, tautan atau hyperlink dibuat menggunakan elemen <a> (anchor). Tautan memungkinkan Anda menghubungkan satu halaman atau sumber daya web dengan yang lain. Berikut adalah beberapa konsep yang terkait dengan tautan HTML:

Elemen <a> digunakan untuk membuat tautan atau hyperlink.

Atribut href menentukan URL atau alamat tujuan tautan. Ini bisa merujuk ke halaman web lain, gambar, dokumen, atau bahkan alamat email.

contoh:

<a href="https://www.example.com">Visit Example.com</a>
