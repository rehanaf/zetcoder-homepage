Dasar HTML

Silahkan setiap kode dicoba dan dipraktekan pada kode editor dan browser masing-masing

Pengenalan

HTML (Hyper Text Markup Language) adalah sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser Internet.

- Wikipedia

HTML adalah bahasa markup yang kemudian diterjemahkan oleh browser menjadi halaman

atau

HTML adalah struktur halaman web

- Rehanaf

Membuat Halaman

Ini adalah contoh halaman HTML

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Belajar Html</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>Ini adalah sebuah paragraf</p>
  </body>
</html>

Hasilnya akan seperti ini

Pembahasan

<!doctype html>

Untuk memberitahu browser kalau dokumen ini html5

<html lang="en">

Penandai awal dan akhir dari dokumen HTML. Atribut lang menunjukkan bahasa halaman (dalam contoh ini, bahasa Inggris). Setiap halaman harus menggunakannya

<head>
  ...
<head>

Mengandung informasi-informasi mengenai dokumen seperti karakter set, viewport, dan judul halaman.

<meta charset="UTF-8">

Menyediakan informasi mengenai karakter set dan konfigurasi tampilan halaman.

<title>Belajar Html</title>
Menentukan judul halaman yang akan ditampilkan di tab browser. 
<body>
  ...
</body>
Tempat untuk elemen-elemen yang akan ditampilkan di halaman web, seperti teks, gambar, dan formulir. 
<h1>Hello World</h1>
<p>Ini adalah sebuah paragraf</p>

Elemen-elemen ini membentuk konten halaman web seperti judul dan paragraf

Struktur Dasar

Setiap elemen HTML didefinisikan dengan menggunakan tag yang diletakkan di antara tanda kurung sudut, seperti <tag>kontennya</tag>. Contohnya, <p> untuk paragraf dan <h1> untuk judul level 1. Elemen-elemen ini membantu browser dalam menampilkan halaman web dengan benar.
