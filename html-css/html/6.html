List

Dalam HTML, terdapat tiga jenis dasar dari list, yaitu Ordered Lists (List Terurut), Unordered Lists (List Tidak Terurut), dan Definition Lists (List Definisi). Berikut adalah penjelasan singkat untuk masing-masing jenis list:

Ol - Ordered List

Digunakan untuk membuat daftar yang diurutkan dengan nomor atau huruf.

contoh:

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Ul - Unordered List

Digunakan untuk membuat daftar yang tidak diurutkan dengan simbol atau ikon (biasanya bulletpoints).

contoh:

<ul>
  <li>Item A</li>
  <li>Item B</li>
  <li>Item C</li>
</ul>

Dl - Definisi List

Digunakan untuk membuat daftar definisi, dengan pasangan elemen <dt> (term) dan <dd> (definition).

contoh:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
