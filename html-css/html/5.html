Gambar

Elemen <img> (Image) pada HTML digunakan untuk menampilkan gambar dalam halaman web.

<img src="gambar.jpg" alt="Deskripsi Gambar">

Atribut src wajib ada dan menunjukkan lokasi atau sumber gambar yang akan ditampilkan. Ini bisa berupa URL eksternal atau path file lokal.

Atribut alt memberikan teks alternatif yang akan muncul jika gambar tidak dapat dimuat. Selain itu, ini juga penting untuk keperluan aksesibilitas, membantu pembaca layar memahami konten gambar.
